> A decentralized, offline-first early warning system that transforms ordinary smartphones into a distributed network of edge-AI weather sensors and rescue beacons.

> Instead of relying solely on top-down satellite data (which fails when internet access goes down during a storm), AeroMesh AI uses a bottom-up approach. It gathers localized data, processes it on the device, and shares it via a peer-to-peer network.
